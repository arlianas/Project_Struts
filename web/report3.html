ΑΝΑΦΟΡΑ ΕΡΓΑΣΙΑΣ PROJECT Struts

Το Struts είναι ένα Java MVC framework που έχει δημιουργηθεί για να κάνει πιο εύκολη την ανάπτυξη web εφαρμογών στην πλατφόρμα J2EE. Περιλαμβάνει ένα σύνολο βιβλιοθηκών και αξιοποιεί ήδη υφιστάμενες τεχνολογίες όπως τα servlets, jsp pages, java beans για τη δημιουργία μιας ολοκληρωμένης web εφαρμογής.

Στη συγκεκριμένη εργασία μας έχει ανατεθεί να αναπτύξουμε περαιτέρω την εφαρμογή του Project 3 χρησιμοποιώντας το Struts framework. Η εφαρμογή είναι ένα διαδικτυακό κατάστημα στο οποίο ο χρήστης έχει τη δυνατότητα να αγοράσει κάποια προϊόντα και παράλληλα να εμφανίζεται ανά πάσα στιγμή η συνολική τιμή των προϊόντων που έχει επιλέξει. Απαιτείται επίσης ακόμα και αν ο χρήστης για κάποιο λόγο διακόψει τη σύνδεση, σε επόμενο session να γνωρίζει τι είχε επιλέξει να αγοράσει. Αυτό επιτυγχάνεται με τη χρήση cookies η οποία έγινε και στο προηγούμενο project.

Πιο αναλυτικά: Χρησιμοποιούμε αυτούσιο το προηγούμενο project και συγκεκριμένα τα σημεία που αφορούν την εμφάνιση των τιμών των προϊόντων (Java Beans και Cookies),την εμφάνιση της σελίδας με την αναφορά του νέου project(Context Listener) καθώς και το κατέβασμα του κώδικα που χρησιμοποιήθηκε για την ανάπτυξη της εφαρμογής(Servlets). Αυτή τη φορά όμως προσθέσαμε και μια επιπλέον λειτουργία. Η welcome page είναι μια σελίδα που περιλαμβάνει μια login form όπου ο χρήστης εισάγει το username του και το e-mail του και αφού τα εισάγει σωστά προωθείται στη σελίδα με τα προϊόντα, όπου εμφανίζονται και τα εισαγόμενα στοιχεία καθώς και ένα μήνυμα επιβράβευσης. Σε περίπτωση εισαγωγής εσφαλμένων στοιχείων εμφανίζεται μήνυμα λάθους και ο χρήστης δεν προωθείται στην τελική σελίδα μέχρις ότου εισάγει τα σωστά. Για αυτή τη διαδικασία χρησιμοποιήσαμε την τεχνολογία Struts. Αυτή περιλαμβάνει τα εξής βασικά στοιχεία: Δύο αρχεία με κλάσεις Java, ένα το οποίο έχει μια κλάση με getters και setters για τα στοιχεία που εισάγονται και ένα άλλο που περιλαμβάνει μια κλάση που αναλαμβάνει να προωθήσει την απαιτούμενη πράξη(action), δηλαδή να μας προωθήσει στη σωστή σελίδα. Για να επιτευχθεί αυτό πρέπει να ορίσουμε τις κατάλληλες ενέργειες και τα κατάλληλα mappings και σε ένα άλλο αρχείο stouts.xml το οποίο χρησιμοποιείται για τα Struts. Στην τελική σελίδα μας μπορούμε να χειριστούμε τα strouts με τρόπο παρόμοιο με τα java beans και να εμφανίσουμε τις πληροφορίες που θέλουμε, εν προκειμένω τα εισαγόμενα στοιχεία του χρήστη.

Για την υλοποίηση της εργασίας χρησιμοποίησα το IDE NetBeans το οποίο διέθετε το πλαίσιο Struts για τη δημιουργία web application και με βοήθησε ιδιαιτέρως η αναζήτηση παραδειγμάτων χρήσης Struts στο διαδίκτυο. Θα ήθελα να τονίσω ότι με τη χρήση Struts θεωρώ ότι αυξήθηκε η πολυπλοκότητα της εφαρμογής(μεγαλύτερος χρόνος για deployment) αν και πρακτικά θεωρώ ότι δουλεύει καλύτερα.
